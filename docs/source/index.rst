Welcome to bcrpy's documentation!
===================================

.. image:: ../img/bcrpy.png
  :width: 150
  :alt: bcrpy logo
  :target: https://github.com/andrewrgarcia/bcrpy


Este modulo de Python es un cliente API para la extraccion, consulta y analisis de la base de datos 
`BCRPData <https://estadisticas.bcrp.gob.pe/estadisticas/series/>`_ del
`Banco Central de Reserva del Peru (BCRP) <https://www.bcrp.gob.pe/>`_ el cual trabaja como un wrapper de la 
`API para Desarrolladores del BCRP <https://estadisticas.bcrp.gob.pe/estadisticas/series/ayuda/api>`_

.. image:: ../img/bcrp.png
  :width: 300
  :alt: BCRP logo
  :target: https://www.bcrp.gob.pe/
